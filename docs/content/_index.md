# kumokagi 雲鍵

**Ephemeral secrets management for Go and Python — one identity, any cloud.**

kumokagi fetches your application's secrets at runtime from HashiCorp Vault, AWS Secrets Manager, Azure Key Vault, GCP Secret Manager, or 1Password — without ever writing them to disk, environment variables, or container manifests.
